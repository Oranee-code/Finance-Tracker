# Boilerplate: Fullstack with Sass

## Setup

Finance Tracker App


A **multi-tracker finance app** built with **React + TypeScript + Tailwind CSS + Framer Motion**. This app allows users to manage multiple trackers (e.g., personal pocket, rental property, family budget) and keep track of income and expenses per category. It calculates spending percentages, provides a visual summary via charts, and allows multiple trackers for different purposes.


## Features


- **Multiple Trackers**: Create separate wallets or trackers for different purposes.
- **Add Income & Expenses**: Record amounts, categories, dates, and notes.
- **Category Tracking**: View spending per category with percentages.
- **Dashboard**: See total income, expenses, and balance for each tracker.
- **Charts**: Pie charts or bar charts for visual representation.
- **Login**: Auth0 integration for secure login.
- **History**: View, edit, or delete past transactions.
- **Smooth Animations**: Framer Motion for animated transitions.


## Tech Stack


- **Frontend**: React + TypeScript + Tailwind CSS
- **Animations**: Framer Motion
- **Backend/Database**: SQLite
- **Authentication**: Auth0
- **Charts**: Chart.js or Recharts


## Screens


1. **Login Screen**: Sign in with Auth0.
2. **Home Screen**: List of all trackers with an option to add new trackers.
3. **Tracker Dashboard**: Shows income, expenses, balance, and charts.
4. **Add Income / Add Expense Screen**: Record transactions with category, amount, date, and notes.
5. **Settings**: Edit tracker, delete tracker, dark mode toggle.
